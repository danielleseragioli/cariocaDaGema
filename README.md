Project developed as a **group assignment** for the **Creative Programming** course  
in the **Bachelor’s in Design** program — **IADE**.

---

## About the Project

**Carioca da Gema** is an experimental web page that explores the intersection of  
**design, creative programming, and visual storytelling**, using the web as an expressive medium.

The project was designed to convey cultural and visual concepts through:
- interaction
- motion
- graphic composition
- creative coding

- Explore creative programming in the web context
